Your feedback was submitted successfully. We'll look at it within a few days and try to act on it promptly.

If a change is requested, and we agree that it needs doing, this is what we're likely to do:

  • If it's an obvious fix, we'll fix it as soon as possible. You'll see this occur via a change in the codebase.
  • If it needs more thought and cannot be done immediately, we might translate it into one or more bugs and put them on our bug tracker.

If, within a few days, you don't see any changes made or new issues filed, please get in touch via the discussion group or the bug tracker. This makes it possible for us to respond directly to the issue.
